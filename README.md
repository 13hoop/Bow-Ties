# Bow-Ties
一个小小的BowTie管理的demo，就是管理你的不同颜色的领结，UI和功能如下：
     segumentController － 可选颜色
     name  － 名称
     rante － 评分，从0-5
     times － 佩戴次数
     lastWorn － 上次佩戴的日期
     Favorite － 是否最爱
     wear － 增加佩戴次数和更新佩戴日期
     rate － 评分允许你重新评分
OK，What you have to do now is take this sample data, store it in Core Data and use it to implement the bow tie management functionality.
